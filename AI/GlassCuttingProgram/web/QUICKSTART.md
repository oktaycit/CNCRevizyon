# 🚀 Quick Start Guide - GlassCutting Pro

## Installation (5 minutes)

### Step 1: Open Chrome Extensions
```
1. Open Google Chrome
2. Type in address bar: chrome://extensions/
3. Press Enter
```

### Step 2: Enable Developer Mode
```
- Toggle the "Developer mode" switch (top right)
- It should turn blue when enabled
```

### Step 3: Load the App
```
1. Click "Load unpacked" button
2. Navigate to: AI/GlassCuttingProgram/web/
3. Select the folder
4. Click "Open"
```

### Step 4: Launch the App
```
1. Go to: chrome://apps/
2. Find "GlassCutting Pro" icon
3. Click to launch
```

---

## First Use (3 minutes)

### Load Sample G-Code
```
1. App opens to G-Code tab
2. Sample code is pre-loaded
3. Or click "📂 Load" to open your file
```

### Set Parameters
```
1. Click "Parameters" in left sidebar
2. Set glass dimensions
3. Set cutting speed/pressure
4. Click "✓ Apply"
```

### Run Simulation
```
1. Click "Simulation" in left sidebar
2. Click "▶ Start" button
3. Watch the cutting path animation
4. Adjust speed with slider (1x-20x)
```

## Common Tasks

### 🔓 Open G-Code File
```
G-Code tab → Click "📂 Load" → Select file → Open
```

### 💾 Save G-Code
```
G-Code tab → Click "💾 Save" → Choose location → Save
```

### ▶ Run Simulation
```
1. Load G-code first
2. Go to Simulation tab
3. Click "▶ Start"
```

### ⚙ Change Settings
```
Click ⚙ (top right) → Adjust settings → Click "Save"
```

### 🔌 Connect to Machine
```
Machine tab → Click "🔌 Connect" → Monitor status
```
