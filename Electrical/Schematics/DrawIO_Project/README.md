# GFB60-30RE-S Draw.io Project

Bu klasor, GFB60/30RE-S elektrik semalarinin draw.io tabanli surumunu icerir.

Baslangic dosyasi:
- `GFB60_30RE_S_Electrical.drawio`

Planlanan sayfalar:
1. Ana guc ve 24VDC dagitimi
2. Emniyet zinciri ve STO
3. NC300 ve EtherCAT omurgasi
4. Servo suruculer
5. R1-EC I/O
6. Terminal ve pin plani

Not:
- Bu surum hizli okunur taslak / dokumantasyon odaklidir.
- Gerekirse daha sonra ECAD ortamina tasinabilir.

Gorsel standart:
- AC guc bloklari: acik gri
- 24VDC / control guc: acik mavi
- Safety bloklari: acik sari
- Network / EtherCAT: acik yesil
- I/O ve terminal dagitimi: acik turuncu veya acik mavi
- Referans / not kutulari: yumu satonlu bilgi kutulari
